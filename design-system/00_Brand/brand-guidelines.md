# NIL Optic Brand Guidelines

Last updated: 2026-03-01

---

## Brand Identity

### Name
**NIL Optic** — styled as two words, capitalized. Never "Niloptic", "nil optic", or "NILOPTIC" in body text. "NILOPTIC" is acceptable in logos and display headers only.

### Tagline
**See the Value. Own the Future.**

### Mission
Empower college athletes, agents, and athletic departments with AI-driven NIL valuation intelligence so every deal is fair, transparent, and data-backed.

---

## Logo

### Primary Mark
- Wordmark: **NIL OPTIC** in bold sans-serif with integrated optic lens icon
- The "O" in OPTIC contains the lens/crosshair motif — representing precision, vision, and data focus
- Minimum clear space: 1x the height of the "N" on all sides

### Logo Variants
| Variant | Background | Usage |
|---------|-----------|-------|
| **Primary** | White mark on Black | Hero sections, dark UI |
| **Inverse** | Black mark on White | Light UI, print |
| **Monochrome Red** | Red mark on Black | Accent, special editions |
| **Icon Only** | Lens/crosshair symbol | App icon, favicon, avatars |

### Logo Don'ts
- Never stretch, rotate, or skew the mark
- Never place on busy photo backgrounds without overlay
- Never change the proportions of the lens icon
- Never use colors outside the brand palette

## Voice & Tone

### Brand Voice Attributes
1. **Authoritative** — We speak with confidence backed by data
2. **Precise** — No fluff, every word earns its place
3. **Empowering** — Athletes and stakeholders gain clarity, not confusion
4. **Technical yet Accessible** — Complex NIL data made understandable

### Writing Guidelines
- Lead with data and outcomes, not features
- Use active voice
- Avoid jargon unless audience is domain-expert
- Numbers speak louder than adjectives — show the metric
- Headlines: short, punchy, metric-forward (e.g., "4.2M Athletes. Zero Transparency. Until Now.")

## Imagery & Photography

### Style
- High-contrast, editorial sports photography
- Black and white as default; red accent overlays for emphasis
- Athletes in action — never staged portraits
- Stadium/arena environments preferred over studio

### Treatment
- Duotone overlay: Black base with Red highlights
- Grain texture: subtle at 5-10% opacity
- Vignette: radial from center, subtle
- Never use stock photography that looks generic

## Application Rules

### Dark Mode (Default)
NIL Optic defaults to dark mode. The interface is built for data-dense environments where dark surfaces reduce eye strain and make data visualizations pop.

### Light Mode
Available as user preference. All components must maintain contrast compliance (WCAG AAA) in both modes.

### Motion
- Motion reinforces state and data changes only
- No decorative animations
- Duration: 150ms–220ms standard
- Easing: ease-out for enters, ease-in for exits

### Spacing
- 4px base unit (`--space-unit`)
- 12-column responsive grid
- Favor clarity over density in vertical rhythm
