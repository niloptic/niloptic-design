# NIL Optic Design Library

> Design-only library. No code. Design system, operations playbooks, pitch deck, and product education for NIL Optic.

## What's Inside

| Directory | Contents | Files |
|-----------|----------|-------|
| `design-system/` | 12-layer design system (Brand through Support) | 28 specs |
| `docs/operations/` | Role playbooks, battle board, team scaling, comms, support | 25 docs |
| `docs/full-blitz-index.md` | Master index connecting all deliverables | 1 |
